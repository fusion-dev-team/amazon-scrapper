# amazon-scrapper

## Install packages
`npm i`

## Start
`npm run start`


## How to start
### Default start
```
npm i
npm run start
```
### Start with params
```
npm i 
npm run start -- -q 3 -o output.csv -s 5 -i ./query_result.csv 
params 
-q = quantity page for search product
-o = output file
-s = start from 
-i = input file 
```
